Inner class

2020е№ҙ09жңҲ28ж—Ҙ 305 еӯ— еӨ§жҰӮ 1 еҲҶй’ҹ

еҶ…йғЁзұ»пјҲInner classпјүжҳҜе®ҡд№үеңЁеҸҰдёҖдёӘзұ»дёӯзҡ„зұ»гҖӮжңүд»ҘдёӢдёүзӮ№дҪҝз”ЁеҺҹеӣ пјҡ

  • еҶ…йғЁзұ»ж–№жі•еҸҜд»Ҙи®ҝй—®иҜҘзұ»е®ҡд№үжүҖеңЁзҡ„дҪңз”Ёеҹҹдёӯзҡ„ж•°жҚ®пјҢеҢ…жӢ¬з§Ғжңүзҡ„ж•°жҚ®гҖӮ
  • еҶ…йғЁзұ»еҸҜд»ҘеҜ№еҗҢдёҖдёӘеҢ…дёӯзҡ„е…¶д»–зұ»йҡҗи—Ҹиө·жқҘгҖӮ
  • еҪ“жғіиҰҒе®ҡд№үдёҖдёӘеӣһи°ғеҮҪж•°дё”дёҚжғізј–еҶҷеӨ§йҮҸд»Јз Ғж—¶пјҢдҪҝз”Ё еҢҝеҗҚпјҲanonymousпјүеҶ…йғЁзұ»жҜ”иҫғдҫҝжҚ·гҖӮ

Inner class еҶ…йғЁзұ»

еҶ…йғЁзұ»зҡ„иҜӯжі•жҜ”иҫғеӨҚжқӮпјҢе…Ҳд»Һз®ҖеҚ•иҜӯжі•ејҖе§Ӣи®Іиө·пјҡ

public class Outer {
    public class InnerTool { // еҶ…йғЁзұ»
        public int add(int a,int b){
            return a+b;
        }
    }
}

еңЁдёҠиҝ°дҫӢеӯҗдёӯпјҢclass Outer жҳҜ еӨ–йғЁзұ»пјҢ class InnerToolжҳҜеҶ…йғЁзұ»гҖӮеҰӮжһңжғіиҰҒи°ғз”ЁеҶ…йғЁзұ»йӮЈд№ҲйҰ–е…Ҳеҝ…йЎ»иҰҒе®һдҫӢеҢ–е®ғдё”еҸӘиғҪеңЁ еҶ…йғЁзұ»дёӯ newгҖӮ

public class OuterUtil {
    public class InnerTool { // еҶ…йғЁзұ»
        public int add(int a,int b){
            return a+b;
        }
    }
    private InnerTool tool = new InnerTool();
}

UMLеӣҫ

еҶ…йғЁзұ»е®һзҺ°пјҡ

е®һдҫӢеҢ–пјҡ

public class OuterUtil {
    public class InnerTool { // еҶ…йғЁзұ»
        public int add(int a,int b){
            return a+b;
        }
    }
    private InnerTool tool = new InnerTool();
    public int add(int a,int b,int c){
       return tool.add( tool.add(a,b) , c );
    }
}
