1. Automata: The Methods and the Madness

1.1 基本概念

  1. 字母表 Alphabet：符号（字符）的非空又穷集

    [P.29] An alphabet is a nite, nonempty set of symbols. Conventionally, we use the symbol ∑ for an alphabet.Common alphabets include:

    1. ∑ = {0,1}, the binary alphabet.

    2. ∑ = {a,b,…,z}, the set of all lower-case letters.

    3. The set of all ASCII characters, or the set of all printable ASCII characters.

  2. 字符串 Strings：由某字母表中符号组成的又穷序列

    [P.29] A string (or sometimes a word) is a nite sequence of symbols chosen from some alphabet. For example, 01101 is a string from the binary alphabet ∑ = {0,1}. The string 111 is another string chosen from this alphabet.

  3. 空串 ø

    [P.29] The empty string is the string with zero occurrences of symbols. This string, denoted , is a string that may be chosen from any alphabet whatsoever. This string, denoted ε, is a string that may be chosen from any alphabet whatsoever.

  4. 字符串的长度：字符串中符号所占位置的个数,记做 |w|。若字母表为 $∑$ ，可以 递归定义 为：
    $$
    |w| =
    \begin{cases}
    0& {w=ε}\
    |x|+1& {w=xa}
    \end{cases}
    $$
    其中 $a$ $\in$ $\Sigma$, $w$ 和 $x$ 是 $\Sigma$ 中字符组成的字符串。

    It is often useful to classify strings by their length, that is, the number of positions for symbols in the string. For instance, 01101 has length 5. It is common to say that the length of a string is “the number of symbols” in the string; this statement is colloquially accepted but not strictly correct. Thus, there are only two symbols, 0 and 1 in the string 01101, but there are fivepositionsfor symbols, and its length is 5. However, you should generally expect that “the number of symbols” can be used when “number of positions” is meant.

  5. 字符串 $x$ 和 $y$ 的连接：将首尾相连得到新字符串的运算，记做 $x·y$ 或 $xy$ 同样可递归定义为
    $$
    x · y = \begin{cases}
    x& {y=ε}\
    (x·z)a& {y=za}
    \end{cases}
    $$
    其中 $a$ $\in$ $\Sigma$, $x,y,z$ 都是字符串。

    Let $x$ and $y$ be strings. Then $xy$ denotes the concatenation of $x$ and $y$, that is, the string formed by making a copy of $x$ and following it by a copy of $y$. More precisely, if $x$ is the string composed of $i$ symbols $x = a_1a_2…a_i$ and $y$ is the string composed of $j$ symbols $y=b_1b_2…b_j$ , then $xy$ is the string of length $i+j:xy=a_1a_2…a_ib_1b_2…b_j$.

  6. 字符串 $x$ 的 $n$ 次幂($n\geq 0$), 递归定义为
    $$
    x^n = \begin{cases}
    ε & {n=0}\
    x^{n-1}x &{n>0}
    \end{cases}
    $$
    例如，若 $\Sigma = {a, b}$ ，那么
    $$
    (ba)^2
    = {ba}^1ba
    = {ba}^0baba
    = εbaba
    = baba \
    ba^2 = ba^1a
    =ba^0aa
    =bεaa
    =baa
    $$

  7. 集合 $A$ 和 $B$ 的连接，记为 $A·B$ 或 $AB$, 定义为
    $$
    A·B = {w|w=x·y,\quad x\in A \quad and \quad y \in B}
    $$

  8. 集合 $A$ 的 $n$ 次幂($n\geq0$), 递归定义为
    $$
    A^n = \begin{cases}
    {ε} & n=0 \
    A^{n-1}A & n \geq 1
    \end{cases}
    $$

    If $∑$ is an alphabet, we can express the set of all strings of a certain length from that alphabet by using an exponential notation. We define $∑^k$ to be the set of strings of length $k$, each of whose symbols is in $∑$.

    若，$\Sigma$ 为字母表，则 $\Sigma^n$ 为 $\Sigma$ 上长度为 $n$ 的字符串集合。如果 $\Sigma = {0,1}$ ，有
    $$
    \Sigma^0 = {ε},\quad \Sigma^1={0,1},\quad \Sigma^2={00,01,10,11}, \
    \Sigma^3 = {000,001,010,011,100,101,110,111}
    $$

    and so on. Note that there is a slight confusion between $∑$ and $∑^1$. The former is an alphabet; its members 0 and 1 are symbols. The latter is a set of strings; its members are the strings 0 and 1, each of which is of length 1. We shall not try to use separate notations for the two sets, relying on context to make it clear whether ${0,1}$ or similar sets are alphabets or sets of strings.

  9. 克林闭包(Kleene Closure)：
    $$
    \Sigma^* = \bigcup_{i=0}^{\infty}\Sigma^i \quad = \Sigma^0\cup\Sigma^1\cup\Sigma^2\cup…\Sigma^{\infty}
    $$

  10. 正闭包(Positive Closure)：
    $$
    \Sigma^+ =\bigcup_{i=1}^\infty\Sigma \quad = \Sigma^1 \cup\Sigma^2 …
    $$
    显然
    $$
    \Sigma^* = \Sigma^+ \cup {ε}
    $$

1.2 语言和问题(Languages & Problems)

定义：若 $\Sigma$ 为字母表且 $\forall$$L$$\subseteq$$\Sigma^*$ ,则 $L$ 称为字母表 $\Sigma$ 上的语言

  • 自然语言，程序设计语言等

  • ${0^n1^n|n\geq0}$

  • The set of string of 0`s and 1`s with an equal number of each:

    ${ε,01,10,0011,0101,1100,…}$

  • $\emptyset , {ε}$ 和 $\Sigma^*$ 分别都是任意字母表 $\Sigma$ 上的语言，注意$\emptyset \neq{ε}$

    关于语言

    唯一重要的约束就是所有字母表都是有穷的

    The only important constraint on what can be a language is that all alphabets are nite. Thus languages, although they can have an infinite number of strings, are restricted to consist of strings drawn from one fixed, nite alphabet.

自动机理论中的典型问题

判断给定的字符串$w$是否属于某个具体语言$L$，$w \in L?$

  • 任何所谓问题，都可以转为语言成员性的问题
  • 语言和问题其实是相同的东西

1.3 形式化证明(Formal Proof)：

演绎法(Deductive Proofs)，归纳法(Inductive Proofs)和反证法(Proofs by contradiction)

1.3.1 归纳法

[Example 1.1].若 $x$ 和 $y$ 是 $\Sigma$ 上的字符串，请证明 $|xy| = |x|+|y|$

证明：通过对 $|y|$ 的归纳来证明
$$
\begin{align*}
基础: 当\quad |y| &== 0，即\quad y = \epsilon\
|x\epsilon| &= |x| \qquad &连接的定义
\&=|x| + |\epsilon| \qquad&长度的定义
\end{align*}
$$

$$
递推:假设\quad|y| = n(n\geq0)\quad时命题成立
\那么当\quad |y| = n+1，即 \quad y=wa
\
\begin{align*}
|x(wa)| &= |(xw)a| \qquad &连接的定义
\&=|xw|+1 \qquad &长度的定义
\&=|x|+|x|+1 \quad &归纳假设
\&=|x|+|wa| \qquad &长度的定义
\end{align*}
$$

2. Finite Automata

有穷自动机是有穷自动系统的抽象模型。

有限状态机：Moore Machine, Mealy Machine.

2.1 确定的有穷自动机

确定的有穷自动机由以下几个部件组成

  • 一条输入带
  • 一个读头
  • 一个有穷控制器(Finite Control)

需要表示的部分：状态、输入带、转移规则、初始状态、接受状态。

2.1.1 形式定义

有穷自动机的形式定义，确定的有穷自动机(DFA, Deterministic Finite Automaton)

$A$ 为五元组
$$
A = (Q,\Sigma,\delta,q_0,F)
$$

$Q:$ 有穷状态集

$\Sigma:$ 有穷输入符号集或字母表

$\delta:$ $Q \times E \to Q$, 状态转移函数

$q_0 \in Q:$ 初始状态

$F\subseteq Q:$ 终结状态集或接受状态集

2.1.2 DFA的设计举例

[Example 2.1]设计DFA,在任何由0和1构成的串中，接受焊油01子串的全部串。

有以下3种情况：

  1. [$q_1$]未发现01，即0都还没有出现过；
  2. [$q_2$]未发现01，但刚刚读入的字符是0；
  3. [$q_3$]已经发现了01。

因此DFA, $A$ 的可定义为：
$$
A =({q_0,q_1,q_2},{0,1},\delta,q_0,{q_1})
$$
其中 $\delta$ 为：
$$
\delta(q_0,1)=q0 \quad \delta(q_1,1)=q_2 \quad \delta(q_2,1)=q_1\
\delta(q_0,0)=q2 \quad \delta(q_1,0)=q_1 \quad \delta(q_2,0)=q_2
$$

**[典型问题]**设计 DFA 使其接受并且仅接受给定的语言 $L$。

[Example 2.2]若 $\Sigma={0,1}$, 给出接受全部含有偶数个0和偶数个1的串。

$L = $ ${$$w|w$ has both an even number of 0’s and an even number of 1’s $}$

在这个DFA中，需要同时数0和1的个数，因此需要2个模来记数。因此有如下4种状态：

$q_0:$ 迄今为止看到0的个数和1的个数都是偶数；

$q_1:$ 迄今为止看到0的个数是偶数，而1的个数是奇数；

$q_2:$ 迄今为止看到1的个数是偶数，而0的个数是奇数；

$q_3:$ 迄今为止看到0的个数和1的个数都是奇数。

其中，状态 $q_0$ 既是初始状态也是唯一的接受状态。在 $q_0$ 状态时，读取的0和1的个数都是0个(0是偶数)又满足设计需求，既偶数个0和偶数个1的串。

因此，得出 $A$ 为：
$$
A=({q_0,q_1,q_2,q_3},{0,1},\delta,q_0,{q_0})
$$

2.1.3 扩展转移函数与DFA的语言

扩展转移函数 Extending the Transition Function to Strings

定义：拓展 $\delta$ 到字符串中，定义扩展转移函数$\hat\delta:Q \times \Sigma^* \to Q$ 为
$$
\hat\delta(q,w)=
\begin{cases}
q & w=\epsilon \
\delta(\hat\delta(q,x),a) &w=xa
\end{cases}
$$

当前状态是 $q$ 输入是 $w$, 如果 $w$ 是空串，则为 $q$ 本身

其中 $a\in \Sigma, w,x \in \Sigma^*.$

那么，当 $w=a_0a_1…a_n$ ，则有
$$
\hat\delta(q,w) = \delta(\hat\delta(q,a_0a_1…a_n-1),a_n)
$$
[Example 2.3]接受全部含有 01 子串的 DFA，$\hat\delta$ 处理串 0101 的过程

![image-20230707144613029](/Users/fraserbu/Library/Application Support/typora-user-images/image-20230707144613029.png)
$$
\begin{align*}
\hat\delta(q_0,0101) &= \delta(\hat\delta(q_0,010),1)\
&=\delta(\delta(\hat\delta(q_0,01),0),1)\
&=\delta(\delta(\delta(\hat\delta(q_0,0),1),0),1)\
&=\delta(\delta(\delta(\delta(\hat\delta(q_0, \epsilon),0),1),0),1)\
&=\delta(\delta(\delta(\delta(q_0,0),1),0),1)\
&=\delta(\delta(\delta(q_1,1),0),1)\
&=\delta(\delta(q_2,0),1) \
&= \delta(q_2,1)=q_2
\end{align*}
$$
