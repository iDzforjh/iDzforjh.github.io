Hands on Linux operation system for Centos 8

Linux notebook for CentOS 8

Linux 内核版本：version: r.x.y

（2.6之前的版本号含义）

​ r：目前发布的kenel主版本

​ x：偶数为稳定版本；奇数为开发版本

​ y：错误修复次数

（2.6～3.0版本）：

​ y：随新版本发布而增加

​ z：bug修复、安全更新、添加新特性和驱动次数增加

（3.0～ ）：

​ 不再使用奇偶数代表开发与稳定版本，0为默认稳定版

Shell命令

基础命令

命令名称 [-选项] [参数]

区分大小写

系统关机命令

halt             //系统正常关机
reboot         //重新启动
poweroff     //系统关机后关闭电源

登陆提示信息含义：

[root@localhost~]#
  1. root：当前用户的账号
  2. localhost：主机名
  3. ～：用户当前所在目录
  4. #：root用户的命令提示符
  5. $：普通用户的命令提示符

ls (list) 命令

ls -a
ls -l
ls -i
ls -d

-a：列出所有项

-l：以长格式输出文件的详细信息

-i：显示文件的索引节点号

-d：只列出目录本身的信息

man命令

man who

查看 who命令的使用手册

data命令

显示系统当前的日期和时间

who命令

who -r
who -q

-r：显示系统当前的运行级

-q：显示登录系统的用户名称和在线总人数

cal命令

日历命令

cal
cal 2019
cal 8 2019

uname命令

显示当前操作系统的相关信息

uname -r
uname -n
uname -v
uname -i

-r：显示操作系统的发行版本号

-m：显示所用机器类型

-v：显示操作系统的版本

-i：显示所需硬件平台

wc命令

wc [lw] [-c] 文件名

输出格式：行数、字数、字节数、文件名

wc -l f1
wc deom demo1 demo2
  1. 统计文件f1的行数
  2. 统计三个文件的信息

扩充

通配符

  1. *：表示任意长度的任何字符
  2. ?：表示任意的一个字符
  3. [] - !：[]、-：表示指定一个符号的取值范围，使用!表示相反含义

特殊符号

  1. ;：隔开多条命令，使他们可以连续执行
  2. &：指令当前命令在后台运行
  3. \：转译符号

输入、输出重定向

  1. 输出重定向和附加输出重定向

    1. 输出重定向：命令 > 重定向文件（覆盖原文件）
    2. 附加输出重定向： 命令 >> 重定向文件（在指定文件末尾追加命令效果）
    date > record
who >> record

cat命令

cat [选项] 文件名

建立小型文件

cat > 文件名
echo cat1 is not a cat1！
# ctrl + d 关闭

文件合并

cat file1 file2 > file3

管道 |

多命令集合，完成复杂功能

more命令

分屏显示文件内容

ls -al /erc|more

vim 文本编辑器

编译器自动：

vim [选项][文件名]

命令模式：

i：从当前光标位开始输入字符
I：在当前行首，开始输入字符
a：从当前光标的下一个位置开始输入字符
A：移动到当前行尾，开始输入字符
o：光标所在行下，插入一行
O：光标所在行上，插入一行

gcc编译器

gcc [选项] [文件名]

源代码文件要求带扩展名（*.c）

可执行代码默认保存在 a.out文件中

常用选项：

-ansi：只支持ANSI标准的c语法
-c：生成目标文件（*.o）
-o [file name]：指定可以执行文件名
-O[level]：进行优化

权限

Linux 用户分类：

  1. 文件所有者(owner)
  2. 同组用户(group)
  3. 其他用户(other)

访问权限：

ls -l 查看文件权限

  • 读取权限
  • 写入权限
    • 文件：修改文件内容的权限
    • 目录：删除、添加和重命名目录内文件的权限
  • 执行权限

访问权限表示法：

  1. 字幕表示法

    1. r：读
    2. w：写
    3. x：可执行或查找
    4. -：无权限

  2. 数字表示法

    相应权限位，有权限为1；无权限为0（八进制）

    • 文件属主：读写权限 7（111）
    • 同组用户：执行权限 5（101）
    • 其他用户：无权限 4（100）

权限管理

ls -l        #查看权限
chmod        #修改访问权限
chown        #修改文件所有者、所属组群
chgrp        #修改文件所属组群

chmod 命令

chmod [字母模式] [文件名]
chmod [数字模式] [文件名]

chmod g+x, o-r [file name]
chmod 670 [file name]

chown 命令

chown [user] [file name]                #修改文件所有者
chown [user:user] [file name]        #修改文件所有者和组群

chgrp 命令

chgrp [group name] [file name]     #修改所属组
